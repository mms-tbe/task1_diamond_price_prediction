# task1_diamond_price_prediction
Этот проект направлен на разработку модели машинного обучения для прогнозирования цен на бриллианты на основе их характеристик.
## Описание проекта

В рамках этого проекта мы работаем с классическим набором данных, содержащим информацию о почти 54,000 бриллиантах. Мы выполняем следующие шаги:

1. Загрузка и исследование данных
2. Проведение разведочного анализа данных (EDA)
3. Разделение данных на обучающую и тестовую выборки
4. Построение базовой модели (линейная регрессия)
5. Оценка производительности модели
6. Анализ важности признаков
7. Визуализация результатов

## Установка

Для запуска этого проекта вам понадобится Python 3.7 или выше. Установите необходимые зависимости с помощью:

```
pip install -r requirements.txt
```

## Получение данных

1. Перейдите на страницу набора данных на Kaggle: [https://www.kaggle.com/datasets/shivam2503/diamonds](https://www.kaggle.com/datasets/shivam2503/diamonds)
2. Если у вас нет аккаунта на Kaggle, зарегистрируйтесь.
3. Нажмите кнопку "Download" в правом верхнем углу страницы.
4. Согласитесь с правилами использования данных, если потребуется.
5. Сохраните файл `diamonds.csv` в корневую директорию проекта.

Примечание: Если у вас возникнут проблемы с доступом к данным на Kaggle, поищите альтернативные источники этого набора данных на GitHub или других платформах для обмена данными.

## Использование

1. Убедитесь, что файл 'diamonds.csv' находится в корневой директории проекта.

2. Запустите скрипт:

```
python diamond_price_prediction.py
```

## Структура проекта

- `diamond_price_prediction.py`: Основной скрипт для анализа и моделирования
- `diamonds.csv`: Набор данных (не включен в репозиторий)
- `requirements.txt`: Список необходимых Python-библиотек
- `README.md`: Этот файл

## Результаты

После выполнения скрипта вы увидите:
- Визуализации распределений характеристик бриллиантов
- Корреляционную матрицу
- Метрики производительности модели (MAE, MSE, R²)
- График важности признаков
- Визуализацию фактических vs предсказанных цен
- График остатков
